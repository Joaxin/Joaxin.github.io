Welcome to Hexo!

This post describes how I set up my Hexo Blog step by step.

Hi, All~ Welcome to Hexo!

Hexo is A fast, simple & powerful blog framework.

  • Blazing Fast
    Node.js brings you incredible generating speed. Hundreds of files take only seconds to build.

  • Markdown Support
    All features of GitHub Flavored Markdown are supported. You can even use most Octopress plugins in Hexo.

  • One-Command Deployment
    You only need one command to deploy your site to GitHub Pages, Heroku or other sites.

  • Various Plugins
    Hexo has a powerful plugin system. You can install more plugins for Jade, CoffeeScript plugins.

    This is my very first post.

How to Install

If you want to hug with the hexo friendly, you do need to have a couple of other things installed first (i.e. Git and Node.js) listed as below:

  • Git

    Windows: Git For Windows
    Mac: Homebrew, MacPorts or installer.
    Linux : sudo apt-get install git-core or sudo yum install git-core

  • Node.js

    Node.js CN

Then, after your computer environment successfully has these tools, congratulations! Just install Hexo with npm, but if you in china , as you know, you need some further steps to get quick access network connections.

In china, alternatively you can use taobao mirror for better experience.
npm install -g cnpm --registry=https://registry.npm.taobao.org
