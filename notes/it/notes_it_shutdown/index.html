电脑关机

Created2017-11-03|Updated2019-11-27|GardenToolsIT
|Word count:525|Reading time:2min|Post View:

🚀目标：了解 Windows电脑的关机方式

Then now, let’s find out…

让我们开始吧…

物理

  1. 台式机机箱上的关机键/笔记本上的电源按钮。

    一般短按关机(笔记本可从电源选项中自主设置)，长按强制关机。

  2. 笔记本-》电源选项 -》选择关闭盖子的功能 , 关机

  3. 任意方式的突然断电(Power Off )

  4. 电脑坏掉了

按钮

  1. 开始菜单(Start Menu)的关机按钮

  2. Ctrl + Alt + Delete 界面右下角的关机键

  3. Alt + F4 的windows对话框(Shut Down Windows dialog box)

    选择关机, 不过先要显示桌面(Win+D)。

  4. win10 最左下角右击，选择众多功能里的关机， 或者快捷键按下Win+X, 打开强大的 Win+X 菜单

  5. win10 按下Windows+C后，面板里的关机按钮

  6. win10的锁屏界面(Lock Screen)的关机按钮

  7. win10优雅的滑动关机功能，打开SlideToShutdown.exe即可(下滑关机, 上滑取消)
