Strings

Words are very important to humans — they are a large part of how we communicate. Since the Web is a largely text-based medium designed to allow humans to communicate and share information, it is useful for us to have control over the words that appear on it.

Strings are really simple—they are just pieces of text. When you give a variable declared enclosed in quotes, the browser will treat the variable as a string, otherwise, JavaScript will try to interpret it as another variable name.

You can use square bracket notation to retrieving a specific string character and indexof and slice() substr() substring() to find a substring inside a string and extracting it.

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/substring

The string methods toLowerCase() and toUpperCase() take a string and convert all the characters to lowercase or uppercase respectively. This can be useful for example if you want to normalize all user-entered data before storing it in a database.

You can replace one substring inside a string with another substring using the replace()method.

Note that if youwant to get the updated value reflected in the helloword variable, you’d have to actually write like this: helloword = helloword.replace('Hello','Hihi');
