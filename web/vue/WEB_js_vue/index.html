Vue.js V3

Created2018-03-19|Updated2020-10-27|WebJS
|Word count:1.5k|Reading time:8min|Post View:

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架，这里记录一下学习Vue.js Ver 3.0 的笔记。

安装(Install)

  1. 从CDN服务器加载安装
  1. npm安装
  1. CLI安装

在Vue2中我们可以用命令，vue init webpack my-project来初始化vue项目，其中用到了webpack，但这里尤大推荐使用Vite（github：https://github.com/vitejs/vite)。

Vite 是一个由原生 ESM 驱动的 Web 开发构建工具。在开发环境下基于浏览器原生 ES imports 开发，在生产环境下基于 Rollup 打包。

具体参见这篇zhihu专栏：@牧遥
