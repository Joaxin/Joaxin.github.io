CSS provides four special universal property values for specifying inheritance:
  • inherit
  • initial
  • unset
  • revert
    • Sets the property value applied to a selected element to be the same as that of its parent element. Sets the property value applied to a selected element to be the same as the value set for that element in the browser's default style sheet. If no value is set by the browser's default style sheet and the property is naturally inherited, then the property value is set to inherit instead. Resets the property to its natural value, which means that if the property is naturally inherited it acts like inherit, otherwise it acts like initial. Reverts the property to the value it would have had if the current origin had not applied any styles to it. In other words, the property's value is set to the user stylesheet's value for the property (if one is set), otherwise, the property's value is taken from the user agent's default stylesheet.
