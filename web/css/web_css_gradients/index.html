CSS - Background Gradients

Created2017-03-17|Updated2018-05-21|WebCSS
|Word count:1.1k|Reading time:6min|Post View:

There are another set of available values for background-image ‚Äî color gradients, which are smooth color transitions across a background.

There are two types of gradient available at the moment:

  • linear gradients (that go from one line straight across to another)
  • radial gradients (which radiate out from a single point)

Background Gradients

A linear gradient is created by passing in a linear-gradient() function as the value of a background-imag property. At a minimum, the function needs to take three parameters separated by commas ‚Äî the direction the gradient should be going in across the background, the color at the beginning, and the color at the end. This gradient would run from top to bottom, starting as orange at the top, and transitioning smoothly to yellow at the bottom. You can use keywords to specify the direction (to bottom/180deg/-180deg, to right, to bottom right, etc.), or degree values (0deg is equivalent to top, 90deg is equivalent to to right, up to -360/360deg, which would be equivalent to to top again).

Radial Gradients

The radial-gradient() CSS function creates an image consisting of a progressive transition between two or more colors that radiate from an origin. Its shape may be a circle or an ellipse.

Value Description
shape Defines the shape of the gradient. Possible values:ellipse (default) circle
size Defines the size of the gradient. Possible values:farthest-corner (default) closest-side closest-corner farthest-side
position Defines the position of the gradient. Default is ‚Äúcenter‚Äù
start-color, ‚Ä¶, last-color Color stops are the colors you want to render smooth transitions among. This value consists of a color value, followed by an optional stop position (a percentage between 0% and 100% or a length along the gradient axis).
