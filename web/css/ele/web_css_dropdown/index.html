这个时候我们发现两个问题，一个是下拉菜单的位置问题，另一个是每个导航栏目之间的间距问题。

  1. 我们在给li标签添加diaplay:inline-block属性后，li标签并排显示会产生3~5px的间距（跟字体大小有关）,这是因为浏览器此时把换行符当成了一个字符，从而产生了间距。解决方案有：

    • 标签不换行(考虑到代码可读性，这样做显然不可取)

    • <li>标签左间距设置-5px；

    • <li>标签不闭合，这时会自动添加闭合，就不会有换行了。

  2. 下拉菜单的位置问题可由position: absolute;解决。
