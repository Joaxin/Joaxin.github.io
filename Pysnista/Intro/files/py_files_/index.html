Created2017-10-21|Updated2019-11-28|PythonIntroduction
|Word count:1.2k|Reading time:6min|Post View:

When you’re working with Python, reading and writing files are very useful when you need to read data or save data.

The first thing you’ll need to do is recognize the the mode attribute of a filein python here.

mode
r open in read-only mode, which is the default rb/r+
w write-only mode for rewriting the contents of a file, create a new file if it doesn’t exist. wb/w+/wb+
a append mode for adding new content to the end of the file, create a new file if it doesn’t exist. ab/a+/ab+
b a mode opens it in binary mode, which is used for non-text files (such as image , pdf, sound and video files) /
+ read & write mode /
